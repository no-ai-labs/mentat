멘타트 시스템 개발 일지

2024년 1월 15일
오늘은 멘타트 Vue 기반 프론트엔드 프로젝트의 DataBlock 기능을 구현했다.
파일 업로드 시스템을 완성하여 CSV, TSV, JSON, TXT 파일을 지원하게 되었다.

주요 성과:
- 드래그 앤 드롭 파일 업로드 구현
- 파일 형식 자동 감지 및 파싱
- 구조화된 데이터 미리보기 기능
- JSON 자동 포맷팅 및 구조 분석
- CSV/TSV 테이블 형태 미리보기

기술 스택:
- Vue 3 Composition API
- TipTap 에디터
- Vue Flow (그래프 에디터)
- Tailwind CSS

다음 단계:
- XLSX 파일 파싱 기능 완성
- 스프레드시트 편집 기능 통합
- 데이터 내보내기 기능 개선

드리머와 함께 만드는 멘타트 시스템이 점점 더 강력해지고 있다!
AI와 인간이 협력하여 창의적인 작업을 할 수 있는 플랫폼을 만들어가고 있다.
